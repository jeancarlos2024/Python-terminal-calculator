# Calculadora em Python (CLI)

Aplicação criada para praticar lógica de programação, controle de fluxo e organização de código, simulando uma calculadora operada via terminal.

# Funcionalidades

    • Menu interativo
    • Operações matemáticas básicas (soma, subtração, multiplicação e divisão)
    • Validação de entrada de dados
    • Tratamento de divisão por zero
    
# Competências Desenvolvidas

    • Modularização com funções
    • Controle de fluxo estruturado
    • Validação e tratamento de dados
    • Organização e legibilidade de código
    
# Objetivo

Consolidar fundamentos de programação e fortalecer a base técnica necessária para evoluir para projetos mais estruturados, incluindo persistência de dados e integração com banco de dados.

# Execução

Python terminal calculator.py
